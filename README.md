1. メールアドレス、パスワード、生成する体温範囲を入力

```python
#              _        ____ _____  #
#   __ _ _   _| |_ ___ | __ )_   _| #
#  / _` | | | | __/ _ \|  _ \ | |   #
# | (_| | |_| | || (_) | |_) || |   #
#  \__,_|\__,_|\__\___/|____/ |_|   #

# consts
LOGIN_ID = "" # メールアドレスを入力
LOGIN_PW = "" # パスワードを入力
BODYTEMP_MIN = # 体温の下限を入力
BODYTEMP_MAX = # 体温の上限を入力
```

2. `pip3 -r requirements.txt`

3. autobt.pyを実行
